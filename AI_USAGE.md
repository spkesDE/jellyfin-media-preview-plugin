# AI Assistance Disclosure

This project uses AI-assisted development tools as part of the development workflow.

AI assistance is used as a supporting tool, mainly for:

- analyzing Jellyfin Web HTML, CSS, and DOM structure
- reviewing possible edge cases
- assisting with refactoring existing code
- suggesting TypeScript structure and module boundaries
- improving documentation and manual test checklists
- improving wording, spelling, grammar, and readability
- helping with clearer issue responses, release notes, and project communication

AI-generated output is not accepted blindly.

The project is maintainer-led. Architecture decisions, debugging, live testing, release decisions, and final code review are handled by the maintainer.

In particular:

- debugging is performed manually by the maintainer
- runtime behavior is tested and validated in a real Jellyfin environment
- the TypeScript structure and project direction are maintainer-defined
- AI suggestions are reviewed, adapted, or rejected before being merged
- the maintainer remains responsible for the final code and releases

AI assistance is also used to support written communication. This includes improving wording, spelling, grammar, and readability in documentation, release notes, issue responses, and other project text.

This project should not be considered a fully AI-generated or "vibe-coded" codebase. AI is used as a development aid, not as an autonomous maintainer.
